# 百日营组长工具 · 使用说明书（小白版）

> **看这篇就够了**。从下载软件到录入第一份作业，手把手一步步来。

---

## 第一步：下载软件

你需要一个叫 **Trae** 的软件（读作"吹"），它是你和 AI 助手对话的窗口。

### 怎么下载？

1. 打开电脑浏览器（随便哪个都行：Edge、Chrome、Safari）
2. 在顶部的网址栏输入：`www.trae.ai`
3. 进入网站后，点那个大大的 **「下载」** 按钮
4. 下载完成后，双击安装文件，一路点「下一步」直到完成
5. 安装好了，桌面上会出现一个蓝色的图标，双击打开它

```
  ┌─────────────────────────────────────┐
  │  ① 打开浏览器                        │
  │  ② 输入 → www.trae.ai                │
  │  ③ 点「下载」                        │
  │  ④ 安装（一直点下一步）                │
  │  ⑤ 桌面出现蓝色图标 → 打开它！         │
  └─────────────────────────────────────┘
```

> ⚠️ **注意**：如果下载不了，或者公司电脑不让装软件，请联系你公司的 IT 同事帮忙。

---

## 第二步：准备你的文件夹

你需要在电脑上建一个文件夹，专门放百日营的所有东西。

### 怎么建文件夹？

1. 在电脑桌面空白处，**右键单击**
2. 选「新建」→「文件夹」
3. 给它起个名字，比如：**百日营第1组**
4. 回车确认，搞定！

```
  桌面
   ├── 我的电脑
   ├── 回收站
   ├── 📁 百日营第1组   ← 这就是你刚建的！
   └── ...
```

### 接下来放什么进去？

把你拿到的两个文件，放进这个文件夹：

| 文件 | 干什么用的 |
|------|-----------|
| `agents_template.md` | AI 助手的"工作说明书"，告诉它该干什么 |
| 你的组员名单 | 写在纸上或记在脑子里也行，一会儿要用 |

**怎么放进去？** 直接拖进去就行。就像把照片拖进相册一样。

```
  📁 百日营第1组
   ├── agents_template.md    ← 拖进来
   └── （后面会自动生成其他文件）
```

## 第三步：让 AI 认识你

### 3.1 打开 Trae，选择你的文件夹

打开 Trae 后：

1. 看屏幕左上角，找到「打开文件夹」或者「Open Folder」
2. 点它，在弹出的窗口里，找到你桌面上的「百日营第1组」文件夹
3. 选中，点「确定」

```
  ┌──────────────────────────────────┐
  │  Trae                             │
  │  ┌─────────────────────────────┐  │
  │  │  文件  编辑  查看  …         │  │
  │  │                             │  │
  │  │  📂 打开文件夹  ← 点这里     │  │
  │  │                             │  │
  │  └─────────────────────────────┘  │
  │                                    │
  │  选择文件夹：                      │
  │  ┌─────────────────────────────┐  │
  │  │ 📁 桌面                     │  │
  │  │   📁 百日营第1组  ← 选这个  │  │
  │  └─────────────────────────────┘  │
  └──────────────────────────────────┘
```

### 3.2 修改模板里的信息

在 Trae 左边的文件列表里，找到 `agents_template.md`，点一下打开它。

然后找到下面这几行，把 `{...}` 改成你自己的：

```
找这行 → {组号}     改成 → 1                （你是第几组就写几）
找这行 → {组长名}   改成 → 王小明            （你的名字）
找这行 → {开营日期} 改成 → 2026年5月21日     （你们哪天开营）
找这行 → {组员名单} 改成 → 张三、李四、王五  （你的组员，逗号隔开）
找这行 → {组员人数} 改成 → 9                （有几个人就写几）
```

> 💡 **怎么改？** 就是用鼠标选中 `{组号}` 这几个字，然后直接打字替换掉。

### 3.3 把文件改个名字

改完上面的内容后，把 `agents_template.md` **重命名**为 `agents.md`。

> 右键点文件名 →「重命名」→ 删掉 `_template` → 回车

## 第四步：跟 AI 说第一句话

在 Trae 界面下方的对话框里，输入这句话：

```
请先阅读 agents.md，了解我的项目，
然后帮我创建一份新的百日营跟踪表 Excel。
我的组员有：张三、李四、王五、赵六、钱七
（这里换成你实际的组员名字）
```

然后按回车。AI 就会开始干活了。

## 第五步：日常使用——录入作业

这是你每天最常用的操作。有组员交了作业，你就这样发给 AI：

### 方法 A：直接打字

在对话框里，按这个格式发：

```
日期 姓名 类型 内容

例子：
5/25 张三 本周作业 今天孩子发脾气，我一开始很生气，
后来觉察到是自己小时候被压抑的情绪投射到了孩子身上。
当我蹲下来抱住他，他慢慢地就不哭了。
```

**完整模板**：你就照着这个写
```
X月X日 XXX 每日觉察（或本周作业/每日践行） [把作业原文粘贴在这里]
```

### 方法 B：复制群聊消息

直接把微信群里的作业消息**复制**，**粘贴**到 Trae 对话框，然后补一句：

> "这是XXX的作业，帮我录入"

AI 会自动识别日期、姓名和内容。

### 方法 C：截图

如果你用的是手机版或支持截图的版本，直接**截图**发过去也行。AI 会识别图片里的文字。

## 常见问题（FAQ）

### ❓ Trae 打不开怎么办？

1. 检查电脑有没有联网
2. 重启电脑试试
3. 去官网重新下载安装
4. 还是不行？找你公司 IT 同事或家里懂电脑的人帮忙

### ❓ AI 回复我看不懂怎么办？

直接对 AI 说：**"说人话，用大白话解释"**。

### ❓ 录错了一条数据怎么办？

对 AI 说："刚才那条 XXX 的数据录错了，正确的应该是……"——AI 会帮你改。

### ❓ 我不小心把文件删了怎么办？

别慌。对 AI 说："XX文件丢了，帮我重新生成一份"。只要你之前录的数据在 AI 的聊天记录里，它就能恢复。

### ❓ 能不能同时开多个窗口？

没必要。一个 Trae 窗口就能搞定一切。你只需要这一个软件。

### ❓ 手机能用吗？

目前主要用电脑操作。因为要管理 Excel 文件，电脑更方便。如果你必须用手机，试试 Trae 有没有手机版。

### ❓ 我的文件夹里东西越来越多了怎么办？

不用担心。AI 会自动帮你建一个备份文件夹 `bairiying/`，旧文件会被整齐地收进去。**你不需要手动整理。**

### ❓ 有组员的作业特别长，我怕复制不过来？

没关系。只复制关键的那几段就行。AI 不需要一模一样的原文，能看懂意思即可。当然能全文复制最好。
